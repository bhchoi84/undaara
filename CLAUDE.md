# Project: 운 다아라

## Tech Stack
- Framework: Next.js
- Language: TypeScript
- UI: Tailwind CSS

## coding convention
- Write all components as functional components.
- Function names start with a verb.
- Use try-catch statements for error handling.
- Write concise comments.

## front end command 
- build `npm run build`
- test `npm test`
